# Mustehsan Nawaz — Portfolio Website

## 📁 Folder Structure

```
portfolio/
├── index.html              ← Main HTML file
├── style.css               ← All styling (dark neon theme)
├── script.js               ← All interactivity & animations
└── assets/
    ├── profile.jpg         ← YOUR profile photo (replace this!)
    ├── resume.pdf          ← YOUR resume PDF (replace this!)
    ├── certificates/
    │   ├── c1.jpg          ← Certificate image 1
    │   ├── c2.jpg          ← Certificate image 2
    │   └── c3.jpg          ← Certificate image 3
    └── icons/
        ├── java.png
        ├── c.png
        ├── python.png
        ├── mysql.png
        ├── html.png
        ├── css.png
        └── js.png
```

## 🚀 How to Run

1. Open `index.html` directly in any modern browser — no server required!
2. Or use VS Code with the **Live Server** extension for live reload.

## 🌟 Features

- ✅ Dark Space Neon theme (Purple + Cyan)
- ✅ Custom cursor with hover ring effect
- ✅ Animated background grid + floating orbs + sparkle particles
- ✅ Typewriter animation (5 rotating phrases)
- ✅ Scroll progress bar (top of page)
- ✅ Scroll reveal animations (fade-up, fade-left, fade-right)
- ✅ Animated skill bars
- ✅ Animated stat counters
- ✅ 3D card tilt on hover
- ✅ Certificate modal popup with zoom animation
- ✅ Responsive navbar with hamburger menu
- ✅ Contact form with send animation
- ✅ Back-to-top button
- ✅ Profile image glow + ring pulse animation
- ✅ Glassmorphism UI cards
- ✅ Fully responsive (mobile + tablet + desktop)

## 📌 Tech Stack Used
- HTML5 (semantic)
- CSS3 (Flexbox, Grid, Custom Properties, Keyframe Animations)
- Vanilla JavaScript (ES6+, IntersectionObserver)
- Google Fonts: Orbitron, Syne, JetBrains Mono
- Font Awesome 6 (icons)
